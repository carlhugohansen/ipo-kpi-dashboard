# IPO KPI Dashboard

Interactive dashboard tracking Key Performance Indicators from MD&A sections of S-1/F-1 IPO filings.

## Features

- **9 companies** across Technology, Business Services, and Consumer Discretionary sectors
- **5 views**: Overview, Comparison Charts, Benchmarking Table, Company Deep Dives, Operational KPIs
- **Global filtering** by Sector, Sub-Sector, and IPO Year
- **S-1 definitions**: Company-specific KPI definitions sourced directly from SEC filings
- Built with Chart.js — fully self-contained single HTML file

## Live Dashboard

Visit the live site: [IPO KPI Dashboard](https://carlhugohansen.github.io/ipo-kpi-dashboard/)

## Data Sources

All financial and operational KPIs are extracted from companies' S-1/F-1 registration statements filed with the SEC.
